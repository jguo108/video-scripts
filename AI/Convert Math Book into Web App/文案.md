上期视频，我们聊了怎么利用谷歌的Gemini AI来和孩子一起共创各种有意思的学习APP。但这还没完，今天来看看，怎么把又厚又枯燥的课本，变成生动好玩的学习网站！

我这里有一本DK出版的少儿数学书Super Simple Maths，将近三百页，虽然包含了很多形象的图示，但真的要啃完里面的知识点，还是个不小的挑战。但如果能够把它变成一个孩子可以互动玩耍的网站呢？放在以前，这肯定是个天方夜谈，但有了新的Gemini3，几句简单的Prompts就能全部搞定！

打开谷歌的AI Studio，选择build，提示词分三部分：Who，What，and How。

首先，Who。让Gemini知道自己是谁。"Assume the role of a UI designer, software engineer, and educational software expert。"

接着，What。"Your task is to convert the first four chapters of a math book into an interactive web application for children。“

最后，How。
”
For each chapter, please:
- **Analyze the content, tone, and target age group**​ of the material.
- **Summarize the core mathematical concepts**​ covered.
- **Propose six key ideas for interactive elements**​ (e.g., mini-games, quizzes, or simulations) and develop them into functional, playable components.
- **Implement a modern web design**​ that is visually appealing and age-appropriate.
- **Apply a thoughtful UI/UX strategy**​ using color, typography, and layout that is intuitive and easy for children to navigate.
“

限于篇幅，这里只选取了书里的头四个章节，但原理都是一样的。经过短短几分钟，Gemini就给我们做出了一个Super Simple Maths的学习网站！这里，你可以通过乘除，看到小数点的左右移动；通过树状结构来对数字进行分解；通过筛除法来快速找到所有的素数；通过点亮方块来理解对称的含义；通过拆分来学会计算多边形的内角之和；通过切Pizza来掌握分数的运算；通过调整边长来看到不同形状的面积和周长的变化。有了这些互动小游戏，再配合上书里的内容，孩子的学习兴趣和效率直接翻倍！

这只是Gemini一次过生成的效果，如果对哪里不满意，还可以随时提出要求来进一步修改。真的是太强了！下次，孩子要是书本看不进去，试试这招，真的会有惊喜！
