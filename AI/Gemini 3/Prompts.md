### Snake Math Game

Create a classic snake game with a math twist for kindergarten children. The snake eats strawberries to score points. For every 50 points, prompt the player with an aged appropriate math question. If they answer correctly, continue with the game. Otherwise, show them the right answer before continuing. The overall design should be simple and beautiful. The score is displayed on top.

### Kitchen Explorer

Create an English education game for kindergarten children. The game is set in the kitchen. Following different recipes, players need to find the different ingredients and put them together.

### Volcano Simulator

Create an interactive volcano simulator for kindergarten children. Children can experiment with the simulator to learn the geographical history of volcanos, different types of volcano, how volcano erupts, what is a environmental impact of an eruption etc.
